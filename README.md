# Russian Word Translator

A lightweight React-based dictionary app that translates Russian words into multiple languages using the Yandex Dictionary API.

## Features

- Translate Russian words to English, German, French and Spanish
- Instant lookup with real-time results
- Clean, minimal interface
- Responsive design

## Tech Stack

- React.js
- Yandex Dictionary API
- Vanilla CSS
